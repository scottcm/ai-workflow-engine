# Code of Conduct

## Our Pledge
We commit to maintaining an open, respectful, and collaborative environment.  
This project is intended for learning, exploration, and professional evaluation.

## Our Standards
Examples of behavior that contributes to a positive environment:
- Using welcoming and inclusive language  
- Respecting differing viewpoints and experiences  
- Offering constructive feedback  
- Focusing on what is best for the project  

Examples of unacceptable behavior:
- Harassment or discrimination of any kind  
- Personal attacks or derogatory comments  
- Disruptive or destructive behavior in issues or discussions  

## Our Responsibilities
Project maintainers are responsible for setting the tone and may remove comments, issues, or contributions that violate this Code of Conduct.

## Scope
This Code of Conduct applies within the project repository, issue tracker, discussions, and any other project communication channels.

## Enforcement
Instances of abusive or unacceptable behavior may be reported by opening an issue or contacting the repository owner directly.  
All reports will be reviewed and addressed as appropriate.
